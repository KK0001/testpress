TDUPRESSって?

TDUPRESSとは東京電機大学東京千住キャンパス新聞委員会が発行する学内広報誌です。

体育祭や旭祭、学術研究発表会などの学内行事の取材以外にも、北千住周辺のグルメ記事や、気になる話題の人へのインタビューなど精力的に様々な活動をしています！

発行されたTDUPRESSは校内の掲示板や、メディアセンター等で配布されているので、見かけたら是非手にとってみて下さい！

部員募集中です！！！

新聞委員会は一緒に活動してくれるメンバーを絶賛募集中です！！！

活動内容は撮影、文芸、編集、デザインと様々なことを行うので、もしどれかに少しでも興味アリ！という方は3号館中2階3M223号室に足を運んでみてください！
