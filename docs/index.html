とは?

TDUPRESSとは東京電機大学東京千住キャンパス新聞委員会が発行する学内広報誌です。体育祭、旭祭、そして学術研究発表会をはじめとした学内行事の取材を中心に、北千住周辺のグルメ記事やインタビューなど精力的な様々な活動を心がけています。発行されたTDUPRESSは校内の掲示板やメディアセンター等で配布されています。見かけたら是非、お手に取ってみてください。

私たちは電大学内広報誌のを作っています。

私たちは年間を通し、5種類のTDUPRESSを作成します。
それぞれには"号"が付けられ、電大の行事と紐づく形で作成されます。


  • 新入生歓迎号
    4月

  • 入学式号
    5月

  • 体育祭号
    6月

  • 旭祭直前号
    10月

  • 旭祭号
    12月
